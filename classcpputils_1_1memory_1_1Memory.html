Detailed Description

template<typename T, typename Allocator = allocators::SimpleAllocator<T>>
class cpputils::memory::Memory< T, Allocator >

RAII wrapper for allocator-backed memory of type T.

Owns a contiguous block allocated by the provided allocator and releases it on destruction. Copy is disabled; moves transfer ownership.

Template Parameters
TThe element type managed by this wrapper
AllocatorThe allocator type used to allocate and deallocate memory
