template<int N, typename... TupleArgs>
requires (N > 0)
struct cpputils::testing::Parameterize< N, TupleArgs >

Marks this test to be parameterized.

Allows for value parameterization for a test. Pass in any amount of tuples, each representing a set of arguments to pass to the test. It will execute the test with each set of parameters.
